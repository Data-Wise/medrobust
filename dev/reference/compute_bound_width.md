# Compute Bound Width

Compute the width of the identification bounds.

## Usage

``` r
compute_bound_width(lower, upper, effect_scale = "OR", scale = "absolute")
```

## Arguments

- lower:

  Lower bound

- upper:

  Upper bound

- effect_scale:

  Character string: "OR", "RR", or "RD"

- scale:

  Character string: "absolute" or "relative"

## Value

Numeric width
