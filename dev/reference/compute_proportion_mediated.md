# Compute Proportion Mediated

Compute the proportion of the total effect that is mediated

## Usage

``` r
compute_proportion_mediated(nie, nde, effect_scale = "OR")
```

## Arguments

- nie:

  Natural Indirect Effect

- nde:

  Natural Direct Effect

- effect_scale:

  Character string: "OR", "RR", or "RD"

## Value

Proportion mediated (PM)
