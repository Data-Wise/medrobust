# Print method for medrobust_bounds objects (S3 - Legacy)

Provides a clean, user-friendly display of partial identification
bounds. NOTE: This is a legacy S3 method. The package now uses S7
methods (see s7-methods.R).

## Usage

``` r
# S3 method for class 'medrobust_bounds'
print(x, digits = 3, ...)
```

## Arguments

- x:

  An object of class `medrobust_bounds`.

- digits:

  Integer specifying the number of decimal places. Default is 3.

- ...:

  Additional arguments (not currently used).

## Value

Invisibly returns the input object.
