# Convert sensitivity_region S7 object to list

Convert sensitivity_region S7 object to list

## Usage

``` r
# S3 method for class 'sensitivity_region'
as.list(x, ...)
```

## Arguments

- x:

  A sensitivity_region S7 object

- ...:

  Additional arguments (ignored)

## Value

A list with sn0_range, sp0_range, psi_sn_range, psi_sp_range
