# Compute Bound Midpoint

Compute the midpoint of the bounds (point estimate).

## Usage

``` r
compute_bound_midpoint(lower, upper, effect_scale = "OR")
```

## Arguments

- lower:

  Lower bound

- upper:

  Upper bound

- effect_scale:

  Character string: "OR", "RR", or "RD"

## Value

Numeric midpoint
